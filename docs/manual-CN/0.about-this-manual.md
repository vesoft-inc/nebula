## 格式约定

**Nebula Graph** 尚在持续开发中，本手册也将持续更新。
本手册使用如下语法惯例：

- `等宽字体`

    等宽字体用于表示**命令**，**需要用户输入的命令**及**接口**。

- **加粗字体**

    用于命令以及其他需要用户逐字输入的文字。

- `UPPERCASE fixed width`

    在查询语言中，`保留关键字` 和 `非保留关键字` 均使用大写等宽字体表示。
