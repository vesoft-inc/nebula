# 比较函数和运算符

| 运算符  | 描述 | 
|:----  |:----:|
| =     | 赋值运算符   | 
| /     | 除法运算符  | 
| ==    | 等于运算符  | 
| !=    | 不等于运算符  | 
| <     | 小于运算符   | 
| <=    | 小于或等于运算符   | 
| -     | 减法运算符   | 
| %     | 余数运算符   | 
| +     | 加法运算符   | 
| *     | 乘法运算符  | 
| -     | 负号运算符   | 


比较运算的结果是 _true_ 或 _false_ 。

* ==

等于。String的比较大小写敏感。不同类的值不相同：

```
nebula> YIELD 'A' == 'a';
==============
| ("A"=="a") |
==============
| false      |
--------------

nebula> YIELD '2' == 2;
============
| ("2"==2) |
============
|false     |
------------
```

* &gt;

大于：

```
nebula> YIELD 3 > 2;
=========
| (3>2) |
=========
| true  |
---------
```

* &ge;

大于或等于：

```
nebula> YIELD 2 >= 2;
==========
| (2>=2) |
==========
| true   |
----------
```

* &lt;

小于：

```
nebula> YIELD 2.0 < 1.9;
=======================
| (2.000000<1.900000) |
=======================
| false               |
-----------------------
```

* &le;

小于或等于：

```
nebula> YIELD 0.11 <= 0.11;
========================
| (0.110000<=0.110000) |
========================
| true                 |
------------------------
```

* !=

不等于：

```
nebula> YIELD 1 != '1'
============
| (1!="1") |
============
| true     |
------------
```
