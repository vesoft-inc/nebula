>用户定义变量只能在一次执行中使用（由分号（;）或管道（|）隔开的复合语法，并提交给服务器一起执行）

可暂时将值存储在前一个语法中的自定义变量中，并在另一个语法中使用，从而可将值从一个语法传递到另一个语法。

自定义变量写为`$ * var_name *`，其中变量名`* var_name *`由字母字符组成。目前不建议使用任何其他字符。

请注意，自定义定义的变量仅可在指定会话中执行。在一个语法中定义的自定义变量不能在其他客户端和其他执行中使用，也就是说定义语法和使用它的语法应该一起提交。当客户端发送执行时，变量会自动释放。

用户变量名称区分大小写。建议名称的最大长度为256个字符。

使用自定义变量的一种方法是使用YIELD语法：

```
YIELD $var1 = expr [, $var2 = expr]
```
